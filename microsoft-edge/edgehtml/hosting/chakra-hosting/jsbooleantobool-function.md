# JsBooleanToBool 関数

ブール値 `bool` の値を取得します。  
  
## 構文  
  
```cpp  
STDAPI_(JsErrorCode) JsBooleanToBool(  
   _In_ JsValueRef value,  
   _Out_ bool *boolValue  
);  
```  
  
#### パラメーター  
 `value`  
 変換する値を指定します。  
  
 `boolValue`  
 変換された値を指定します。  
  
## 戻り値  
 操作が `JsNoError` 成功した場合はコード、それ以外の場合はエラー コード。  
  
## 注釈  
 アクティブなスクリプト コンテキストが必要です。  
  
## 要件  
 **ヘッダー:** jsrt.h
