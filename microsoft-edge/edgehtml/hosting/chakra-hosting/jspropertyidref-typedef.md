# JsPropertyIdRef Typedef

プロパティ識別子。  
  
## 構文  
  
```cpp  
typedef JsRef JsPropertyIdRef;  
```  
  
## 注釈  
 プロパティ識別子は、文字列を使用する代わりに JavaScript オブジェクトのプロパティを参照するために使用されます。  
  
## 要件  
 **ヘッダー:** jsrt.h
