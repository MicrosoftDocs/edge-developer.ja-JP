# JsGetRuntime 関数

コンテキストが属するランタイムを取得します。  
  
## 構文  
  
```cpp  
STDAPI_(JsErrorCode) JsGetRuntime(  
   _In_ JsContextRef context,  
   _Out_ JsRuntimeHandle *runtime  
);  
```  
  
#### パラメーター  
 `context`  
 ランタイムを取得するコンテキスト。  
  
 `runtime`  
 コンテキストが属するランタイム。  
  
## 戻り値  
 操作が `JsNoError` 成功した場合はコード、それ以外の場合はエラー コード。  
  
## 要件  
 **ヘッダー:** jsrt.h
