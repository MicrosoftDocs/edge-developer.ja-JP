# JsProjectWinRTNamespace 関数

WinRT 名前空間を投影します。  
  
## 構文  
  
```cpp  
STDAPI_(JsErrorCode)  
   JsProjectWinRTNamespace(  
   _In_z_ const wchar_t *namespaceName  
);  
```  
  
#### パラメーター  
 `namespaceName`  
 投影する WinRT 名前空間。  
  
## 戻り値  
 操作が `JsNoError` 成功した場合はコード、それ以外の場合はエラー コード。  
  
## 注釈  
 アクティブなスクリプト コンテキストが必要です。  
  
 この API は EdgeHTML モードでのみサポートされています。  
  
> [!NOTE]
>  WinRT は、ユニバーサル Windows プラットフォーム (UWP) より前のプラットフォーム名でした。  
  
## 要件  
 **ヘッダー:** jsrt.h
