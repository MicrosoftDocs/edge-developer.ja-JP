# JsHasExternalData 関数

オブジェクトが外部オブジェクトであるかどうかを判別します。  
  
## 構文  
  
```cpp  
STDAPI_(JsErrorCode) JsHasExternalData(  
   _In_ JsValueRef object,  
   _Out_ bool *value  
);  
```  
  
#### パラメーター  
 `object`  
 オブジェクト。  
  
 `value`  
 オブジェクトが外部オブジェクトかどうかを指定します。  
  
## 戻り値  
 操作が `JsNoError` 成功した場合はコード、それ以外の場合はエラー コード。  
  
## 注釈  
 アクティブなスクリプト コンテキストが必要です。  
  
## 要件  
 **ヘッダー:** jsrt.h
