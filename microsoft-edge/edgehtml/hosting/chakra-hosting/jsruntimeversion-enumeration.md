# JsRuntimeVersion 列挙

ランタイムのバージョン。  
  
## 構文  
  
```cpp  
enum JsRuntimeVersion;  
```  
  
## Members  
  
### 値  
  
|名前|説明|  
|----------|-----------------|  
|`JsRuntimeVersion10`|IE10 バージョンを使ってランタイムを作成します。|  
|`JsRuntimeVersion11`|IE11 バージョンを使ってランタイムを作成します。|  
|`JsRuntimeVersionEdge`|実行時にコンピューターに最も高いバージョンが存在するランタイムを作成します。|  
  
## 要件  
 **ヘッダー:** jsrt.h
