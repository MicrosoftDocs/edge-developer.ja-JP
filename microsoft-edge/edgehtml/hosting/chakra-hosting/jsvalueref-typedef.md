# JsValueRef Typedef

JavaScript 値への参照。  
  
## 構文  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## 注釈  
 JavaScript の値は、Undefined、Null、Boolean、String、Number、または Object のいずれかの種類の値です。  
  
## 要件  
 **ヘッダー:** jsrt.h
