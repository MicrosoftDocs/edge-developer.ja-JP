# 0.8.355-インターフェイス IWebView2CreateWebViewCompletedHandler 

[!INCLUDE [deprecation-note](../../includes/deprecation-note.md)]

呼び出し元は、CreateWebView 経由で作成された WebView を受け取るために、このインターフェイスを実装します。
