| [CSP143xx](#csp-codes) | コンテンツセキュリティポリシー | HTTP ヘッダーを`Content-Security-Policy`使ってブロックされるリソース。  |  
| [CSS31xx](#css-codes) | シート | *Web Open Font Format* (woff \) と*埋め込み OpenType* \ (EOT \) フォントソースおよびホストサーバーの問題に関連するエラー。  |  
| [HTML1112-1300](#html-codes) | HTML | これらのほとんどは、*ドキュメントモード*や*互換表示*などの Microsoft Internet Explorer の概念に固有の従来のコードです。  |  
| [HTML1400-1532](#html5-parser-warnings) | HTML5 の解析 | HTML5 パーサーによってスローされた検証の警告。  |

### Windows ランタイムエラー  

 Windows ランタイム \ (WinRT \) Api を使っている場合、Windows ランタイム Hresult から変換された JavaScript エラーが表示されることがあります。  Windows ランタイムが0x80070000 の範囲内にある場合は、下位ビットの16進値を取得して10進数に変換することで、JavaScript エラーに変換されます。  たとえば、HRESULT 0x80070032 は10進値の50に変換され、JavaScript エラーは SCRIPT50 になります。  HRESULT 0x80074005 は10進数値の16389に変換され、JavaScript エラーは SCRIPT16389 ます。
