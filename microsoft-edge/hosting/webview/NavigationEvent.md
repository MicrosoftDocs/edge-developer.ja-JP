# NavigationEvent オブジェクト

ナビゲーションが開始されたときに発生するイベントを表すオブジェクト。

## プロパティ
    
### uri

ターゲットの Uniform Resource Identifier (URI)。

このプロパティは読み取り専用です。

```js
var uri = NavigationEvent.uri;
```

#### プロパティ値
Type: **Domstring**
