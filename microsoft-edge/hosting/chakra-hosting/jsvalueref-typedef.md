# JsValueRef Typedef
JavaScript 値への参照。  
  
## 構文  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## 注釈  
 JavaScript の値は、Undefined、Null、ブール値、文字列、数値、またはオブジェクトのいずれかの型です。  
  
## 要件  
 **ヘッダー:** jsrt
