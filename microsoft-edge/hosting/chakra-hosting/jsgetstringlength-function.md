# JsGetStringLength 関数
文字列値の長さを取得します。  
  
## 構文  
  
```cpp  
STDAPI_(JsErrorCode) JsGetStringLength(  
   _In_ JsValueRef stringValue,  
   _Out_ int *length  
);  
```  
  
#### パラメーター  
 `stringValue`  
 長さを取得する文字列値。  
  
 `length`  
 文字列の長さ。  
  
## 戻り値  
 `JsNoError`操作が正常に完了した場合は、エラーコードは返されません。  
  
## 注釈  
 アクティブなスクリプトコンテキストが必要です。  
  
## 要件  
 **ヘッダー:** jsrt
