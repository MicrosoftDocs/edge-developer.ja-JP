# JsPropertyIdType 列挙
JavaScript プロパティの列挙型を入力します。  
  
## 構文  
  
```cpp  
enum JsPropertyIdType;  
```  
  
## Members  
  
### 値  
  
|名前|説明|  
|----------|-----------------|  
|JsPropertyIdTypeString|JavaScript 文字列プロパティの列挙型|  
|JsPropertyIdTypeSymbol|JavaScript シンボルプロパティの列挙型|
