# JsRuntimeVersion 列挙
ランタイムのバージョン。  
  
## 構文  
  
```cpp  
enum JsRuntimeVersion;  
```  
  
## Members  
  
### 値  
  
|名前|説明|  
|----------|-----------------|  
|`JsRuntimeVersion10`|IE10 バージョンのランタイムを作成します。|  
|`JsRuntimeVersion11`|IE11 バージョンのランタイムを作成します。|  
|`JsRuntimeVersionEdge`|実行時にマシン上に存在する最大バージョンのランタイムを作成します。|  
  
## 要件  
 **ヘッダー:** jsrt
