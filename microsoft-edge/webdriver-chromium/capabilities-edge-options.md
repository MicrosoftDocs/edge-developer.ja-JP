# <span data-ttu-id="c4571-104">Capabilities と EdgeOptions</span><span class="sxs-lookup"><span data-stu-id="c4571-104">Capabilities and EdgeOptions</span></span>  

<span data-ttu-id="c4571-105">Capabilities は、`EdgeDriver` セッションをカスタマイズして構成するために使用できるオプションです。</span><span class="sxs-lookup"><span data-stu-id="c4571-105">Capabilities are options that you may use to customize and configure an `EdgeDriver` session.</span></span>  <span data-ttu-id="c4571-106">`EdgeDriver` セッションを開始するには、「[Microsoft Edge の推進][DrivingEdgeWebDriverIndex]」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="c4571-106">To start an `EdgeDriver` session, navigate to [Driving Microsoft Edge][DrivingEdgeWebDriverIndex].</span></span>  <span data-ttu-id="c4571-107">この記事では、[Microsoft Edge][DownloadEdgeWebDriverIndex] でサポートされているすべての Capabilities と、Capabilities を `EdgeDriver` セッションに渡すための追加の詳細について説明します。</span><span class="sxs-lookup"><span data-stu-id="c4571-107">This article documents all supported capabilities for [Microsoft Edge][DownloadEdgeWebDriverIndex] and additional details for passing the capabilities to an `EdgeDriver` session.</span></span>  

<span data-ttu-id="c4571-108">WebDriver 言語バインディングは、Capabilities を `EdgeDriver` に渡すための手段を提供します。</span><span class="sxs-lookup"><span data-stu-id="c4571-108">WebDriver language bindings provide ways to pass capabilities to `EdgeDriver`.</span></span>  <span data-ttu-id="c4571-109">正確なメカニズムは、[選択した言語バインディング][ChooseLanguageBindingWebDriverIndex]によって異なります。</span><span class="sxs-lookup"><span data-stu-id="c4571-109">The exact mechanism depends on the [language binding you choose][ChooseLanguageBindingWebDriverIndex].</span></span>  <span data-ttu-id="c4571-110">[Selenium][SeleniumMain] では、Capabilities は `EdgeOptions` クラスによって提供されます。</span><span class="sxs-lookup"><span data-stu-id="c4571-110">In [Selenium][SeleniumMain], capabilities are provided through the `EdgeOptions` class.</span></span>  

## <span data-ttu-id="c4571-111">EdgeOptions クラスの使用</span><span class="sxs-lookup"><span data-stu-id="c4571-111">Using the EdgeOptions class</span></span>  

<span data-ttu-id="c4571-112">`EdgeOptions` のインスタンスを作成 します。これには、Microsoft Edge 固有の Capabilities を設定するための便利なメソッドがあります。</span><span class="sxs-lookup"><span data-stu-id="c4571-112">Create an instance of `EdgeOptions`, which has convenient methods for setting Microsoft Edge-specific capabilities.</span></span>  <span data-ttu-id="c4571-113">`EdgeOptions` オブジェクトを設定したら、`EdgeOptions` を `EdgeDriver` コンストラクターに渡します。</span><span class="sxs-lookup"><span data-stu-id="c4571-113">Once you have configured the `EdgeOptions` object, pass `EdgeOptions` into the `EdgeDriver` constructor.</span></span>  

```csharp
var options = new EdgeOptions();
options.UseChromium = true;
options.AddExtensions("/path/to/extension.crx");
var driver = new EdgeDriver(options);
```  

<span data-ttu-id="c4571-114">便利なメソッドがない Capabilities については、`AddAdditionalCapability` メソッドを使います。</span><span class="sxs-lookup"><span data-stu-id="c4571-114">For capabilities that do not yet have a convenient method, use the `AddAdditionalCapability` method.</span></span>  <span data-ttu-id="c4571-115">Capability の名前とそれが受け入れる値の型を知っている必要があります。</span><span class="sxs-lookup"><span data-stu-id="c4571-115">You must know the name of the capability and the type of value it accepts.</span></span>  <span data-ttu-id="c4571-116">完全な一覧を確認するには、「[EdgeOptions オブジェクト](#edgeoptions-object)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="c4571-116">To review the full list, navigate to [EdgeOptions object](#edgeoptions-object).</span></span>  

```csharp
options.AddAdditionalCapability("wdpAddress", "remotehost:50080");
```
